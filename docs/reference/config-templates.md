# Config templates

Scaffold a TOML config file from the `[config: key]` annotations in a usage message - the file the
`docopt(config=...)` mapping is loaded from. See the
[config skeleton](../guides/usage-dsl.md#generate-a-config-skeleton) guide.

::: docopt2.generate_config_template
    options:
      show_root_heading: true
      show_root_toc_entry: true
