# Exceptions

The two exception types docopt2 raises: one for a failed parse of argv, one for a malformed usage.

::: docopt2.DocoptExit
    options:
      show_root_heading: true
      show_root_toc_entry: true

::: docopt2.DocoptLanguageError
    options:
      show_root_heading: true
      show_root_toc_entry: true
